# Portfolio React - ADJOLOKOU Eyabana 

## Description

Ce projet est un portfolio développé avec React dans le cadre de la formation Développeur Web et Web Mobile du CEF.

Le site comprend :

- Une page d'accueil
- Une page Services
- Une page Réalisations
- Une page Blog
- Une page Contact
- Une page Mentions légales
- L'affichage des informations d'un profil GitHub grâce à l'API GitHub

---

## Technologies utilisées

- React 19
- Vite
- React Router DOM
- Bootstrap 5
- Font Awesome
- React Helmet Async
- HTML5
- CSS3
- JavaScript

---

## Prérequis

Avant de lancer le projet, installer :

- Node.js (version 18 ou supérieure)
- npm
- Git (optionnel)

---

## Installation

Cloner le dépôt :

```bash
git clone https://github.com/adjolokoueyabana/cef-cv-react.git
```

Se placer dans le dossier :

```bash
cd cef-cv-react
```

Installer les dépendances :

```bash
npm install
```

---

## Lancement du projet

Démarrer le serveur de développement :

```bash
npm run dev
```

Puis ouvrir :

```
http://localhost:5173
```

---

## Génération de la version de production

```bash
npm run build
```

Les fichiers générés se trouvent dans le dossier :

```
dist/
```

---
